# ProjectSite
Это приложение предсталяет собой сайт про внутреннее устройство компьютера
Этот приложение представляет собой сайт с серверной частью(backend),реализованной на C# и внешней частью(frontend) реализованной на связке javescript+html+css
Это приложение ориентировано на людей, которым интересно внутреннее устройство компьютера и желание в нем разобраться
