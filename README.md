# ProjectSite
Это приложение предсталяет собой сайт для компьютерной периферии
Этот приложение представляет собой сайт с серверной частью(backend),реализованной на C# и внешней частью(frontend) реализованной на связке javescript+html+css
Это приложение ориентировано на людей, которым нужны акксесуары для компьютера или ноутбука, например:наушники,микрофоны,компьютерные мыши,клавиатуры,микрофоны и другие акксесуары
