### Название приложения: PC Architecture Guide

### Описание приложения:
Веб-приложение для изучения внутреннего устройства персонального компьютера.
Пользователи могут просматривать интерактивные схемы, читать подробные описания компонентов (процессор, ОЗУ, видеокарта, материнская плата и другие) и изучать принципы их взаимодействия.
Приложение помогает систематизировать знания об аппаратной части ПК в удобном визуальном формате.

### Примерная целевая аудитория:
Студенты технических специальностей, начинающие IT-специалисты, энтузиасты, желающие разобраться в аппаратном обеспечении компьютера.

### Описание
Веб-приложение для изучения внутреннего устройства компьютера. Интерактивный интерфейс с анимациями и эффектами.

## Функционал

### Главная страница (`/`)
- Три варианта входа (администратор, пользователь, гость)
- Эффект свечения для изображения
- Skeleton-загрузка при инициализации
- Адаптивный дизайн для мобильных устройств

### Страница регистрации и входа  (`/register`,`/login`)
- Форма с полями: имя, email, пароль, подтверждение пароля
- Валидация данных на стороне клиента
- Проверка совпадения паролей
- Минимальная длина пароля (6 символов)
- Перенаправление на страницу входа после успешной регистрации
- Форма авторизации
- Возможность перехода на регистрацию

## Особенности реализации

### Маршрутизация
- Настроена с помощью Vue Router
- Используется история HTML5 (createWebHistory)
- Динамическая загрузка компонентов
### Разделы
- **1.Раздел с главной страницей**-отображает главную страницу
- **2.Раздел с взаимодействием комплектующих и вывод для некоторой из списка для конкретной комплектующей конкретных параметров**;
*Представляет собой навигацию по комплектующим и как они взаимодейтвуют с друг с другом и вывод характеристик для конкретной комплектующей*
- **3.Раздел авторизации входа,регистрации и восстановленем пароля пользователей как гость,администратор и пользовательв**;
*Представляет собой страницу с вводом имени,пароля и email для входа под видом пользователя,гостя или адиминистратора*
*Пользователь должен быть зарегестрирован иначе при попытке входа последует регистрация*
*Пароль можно восстановить*
*Гость может послещать все разделы кроме викторины и личного кабинета*
*Администратор может смотреть списки пользователей и их удалять по необходимости и также везде взаимодействовать,но без прогресса*
- **4.Раздел с лабораторией разгона** -
*Интерактивные слайдеры в реальном времени:Графики температур, стабильности, FPS               
*Рабочая область с несколькими компонентами одновременно* 
*Симуляция тестов с прогресс-барами и результатами*       
*Сравнение «до/после» с визуализацией прироста*
- **5.Раздел с личным кабинетом пользователя**;
  *Представляет собой кабинет с отображаемой почтой и скрытым паролем и также статистику пройденной викторины и прогресс изучения комплектующих*
- **6.Раздел с опросом для пользователей**;
-  *Представляет собой викторину с тематическими вопросами по комплектующим компьютера*
- **7.Раздел с подбором всех комплектующих под сборку персонального компьютера**
    Представляет собой: *интерактивные фильтры совместимости в реальном времени — при выборе каждого компонента система проверяет сокет,
       мощность БП, размеры корпуса, показывает зелёные/красные индикаторы*
  *Рабочая область со всеми категориями комплектующих одновременно — процессор, материнская плата, ОЗУ, GPU,
       накопители, БП, корпус, охлаждение*
  *Симуляция сборки с прогрессом заполнения — визуализация занятых слотов, свободных разъёмов, прогноза
       кабель-менеджмента*
  *Сравнение «бюджет vs производительность» — график распределения бюджета по компонентам, рекомендации по
       оптимизации цены*
- **8.Раздел с BIOS/UEFI - база прошивок для материнских плат с инструкциями по обновлению, предупреждениями о рисках, историей версий, откатом к предыдущей**;
  Представляет собой: *Интерактивная история версий — таймлайн всех версий BIOS для выбранной платы с датами выхода, изменениями,
       стабильностью*
  *Калькулятор рисков обновления — пользователь выбирает текущую и целевую версию, система показывает риск
       (низкий/средний/высокий) с факторами*
  *Симулятор обновления с прогресс-баром — пошаговая визуализация процесса прошивки с предупреждениями на каждом
       этапе*
  *Таблица совместимости CPU ↔ BIOS — динамическая проверка: какая версия нужна для поддержки конкретного
       процессора, можно ли откатиться*

## Модели данных

**User** — `id, username, email, passwordHash, role (Guest/User/Admin), createdAt`

**Component** — `id, name, category (CPU/GPU/RAM/etc.), manufacturer, price, specifications, socket, powerConsumption`

**PCBuild** — `id, userId, name, componentIds[], totalPrice, isCompatible`

**QuizResult** — `id, userId, score, totalQuestions, completedAt`

**QuizQuestion** — `id, question, options[], correctAnswer, difficulty`

**BiosVersion** — `id, motherboardId, version, releaseDate, description, stability, isBeta`

**Motherboard** — `id, name, socket, chipset`

**CpuSupport** — `id, cpuName, biosVersionId, isSupported`

**OverclockProfile** — `id, userId, cpuName, frequency, voltage, temperature, stability, createdAt`

## API Endpoints

###  Auth
| Метод | Endpoint | Описание |
|-------|----------|----------|
| POST | `/api/auth/register` | Регистрация |
| GET | `/api/auth/login` | Вход |
| PUT | `/api/auth/forgot-password` | Восстановление пароля |
| GET | `/api/auth/me` | Текущий пользователь |

| GET | `/api/components` | Список комплектующих |
